Startup Info API

API Documentation & Usage

Welcome to the Startup Info API. You can fetch a list of startups and filter them using various search queries.

Endpoints

  • GET /api/startups - Fetches all startups.
  • GET /api/startups/:id - Fetches a single startup by its ID.

Search Queries

You can add these parameters to the `/api/startups` endpoint:

  • ?search=keyword - Fuzzy search by startup name, industry, description, or founder's name.
  • ?industry=FinTech - Filter by a specific industry.
  • ?country=Nigeria - Filter by country.
